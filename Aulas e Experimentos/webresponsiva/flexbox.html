Introdução ao Flexbox

O Flexbox (Flexible Box Layout) é um modelo de layout CSS que permite alinhar e distribuir espaço entre os itens em um contêiner, mesmo quando seu tamanho é desconhecido e/ou dinâmico.

Propriedades úteis do Flexbox

  • flex-direction: muda a direção do eixo principal (ex: row, column)
  • justify-content: alinha no eixo principal (start, center, space-between, etc.)
  • align-items: alinha no eixo cruzado (start, center, stretch, etc.)
  • flex-wrap: permite quebrar linha quando necessário
  • gap: define espaçamento entre os itens
  • flex-grow: define quanto um item deve crescer
  • align-self: alinha individualmente no eixo cruzado
